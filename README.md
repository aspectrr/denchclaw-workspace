# denchclaw-workspace

An example [OpenClaw](https://openclaw.com) workspace setup — showcasing agent configuration, custom skills, and workspace structure.

## Structure

- `skills/` — Custom agent skills
- `scripts/` — Utility scripts
- `docs/` — Strategy and process documentation
- `AGENTS.md` — Agent definitions
- `IDENTITY.md` — Workspace identity config
- `SOUL.md` — Agent personality/behavior
- `TOOLS.md` — Tool configurations
- `WORKSPACE.md` — Workspace overview
